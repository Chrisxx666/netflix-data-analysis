Netflix Data Analysis Project

Exploring content trends and patterns in the Netflix catalog

Project Overview

This project provides a comprehensive analysis of Netflix's content catalog, examining trends over time, content distribution, and audience targeting through ratings. The analysis is built using Python, pandas, and MySQL, with visualizations created using Plotly and Looker Studio.

The dashboard presents insights on yearly content additions, content type distribution, geographic distribution of content production, and content rating analysis.

View Interactive Dashboard GitHub Repository

Key Insights

Yearly Growth

Netflix content additions have steadily increased year over year, with significant growth after 2015.

Content Types

Movies represent approximately 70% of Netflix's catalog, while TV shows make up the remaining 30%.

Geographic Distribution

The United States leads in content production, followed by India and the United Kingdom.

Content Ratings

TV-MA (mature audience) is the most common rating, indicating Netflix's focus on adult content.

Visualizations

Methodology

Data Pipeline
  1. Data Loading: CSV loading using pandas
  2. Data Cleaning: Handling missing values and standardizing formats
  3. Data Storage: MySQL database for persistent storage
  4. Analysis: Python and pandas for data analysis
  5. Visualization: Plotly for interactive charts and Looker Studio for dashboards

The entire process is automated through a Python pipeline that can be executed with a single command.
